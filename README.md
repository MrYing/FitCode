# 码上健身小程序
## 小程序试水，腾讯2018年高校微信小程序大赛作品。
功能较为简单：用户登陆后通过小程序扫描指定的健身器械二维码，弹出器械的相关入门级锻炼方法，并在用户锻炼后记录锻炼类型。
